Spiralcraft

spiralcraft.loader package

The spiralcraft.loader package is responsible for loading and invoking Spiralcraft applications by resolving code dependencies and constructing an appropriate execution context for the application.

The primary use of the spiralcraft.loader package is when the spiralcraft.loader.Main class is invoked by the spiralcraft-main package to run an application in a newly created Java VM.

Other planned uses of the spiralcraft.loader package involve the invocation of applications in an already running VM on behalf of administration tools.
